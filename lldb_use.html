GDB → LLDB
Command Map

A side-by-side reference of GDB commands and their LLDB counterparts, including built-in GDB-compatibility aliases. Full command names are long, but any unique prefix works — br se instead of breakpoint set.

▶ Execution Commands

Launch a process (no arguments)

GDB
(gdb) run
(gdb) r
LLDB
(lldb) process launch
(lldb) run
(lldb) r

Launch a process with arguments <args>

GDB
(gdb) run <args>
(gdb) r <args>
LLDB
(lldb) process launch -- <args>
(lldb) run <args>
(lldb) r <args>

Launch a.out with arguments 1 2 3 — passing args to the debugger

GDB
% gdb --args a.out 1 2 3
(gdb) run
(gdb) run
LLDB
% lldb -- a.out 1 2 3
(lldb) run
(lldb) run

Launch a.out with arguments 1 2 3 — setting args in the debugger

GDB
(gdb) set args 1 2 3
(gdb) run
LLDB
(lldb) settings set target.run-args 1 2 3
(lldb) run

Launch with arguments in a new terminal window (macOS only)

LLDB
(lldb) process launch --tty -- <args>
(lldb) pro la -t -- <args>

Launch with arguments in existing terminal /dev/ttys006

LLDB
(lldb) process launch --tty=/dev/ttys006 -- <args>
(lldb) pro la -t/dev/ttys006 -- <args>

Set environment variables before launching

GDB
(gdb) set env DEBUG 1
LLDB
(lldb) settings set target.env-vars DEBUG=1
(lldb) env DEBUG=1

Unset environment variables before launching

GDB
(gdb) unset env DEBUG
LLDB
(lldb) settings remove target.env-vars DEBUG
(lldb) set rem target.env-vars DEBUG

Show arguments passed to the program

GDB
(gdb) show args
LLDB
(lldb) settings show target.run-args

Set environment and launch in one command

LLDB
(lldb) process launch -E DEBUG=1

Attach to process ID 123

GDB
(gdb) attach 123
LLDB
(lldb) process attach --pid 123
(lldb) attach -p 123

Attach to process named a.out

GDB
(gdb) attach a.out
LLDB
(lldb) process attach --name a.out
(lldb) pro at -n a.out

Wait for a.out to launch and attach

GDB
(gdb) attach -waitfor a.out
LLDB
(lldb) process attach --name a.out --waitfor
(lldb) pro at -n a.out -w

Attach to remote gdb server on eorgadd:8000

GDB
(gdb) target remote eorgadd:8000

(lldb) gdb-remote eorgadd:8000

GDB
(gdb) target remote localhost:8000
LLDB
(lldb) gdb-remote 8000

Attach to Darwin kernel via kdp on eorgadd

GDB
(gdb) kdp-reattach eorgadd
LLDB
(lldb) kdp-remote eorgadd

Source-level single step in

GDB
(gdb) step
(gdb) s
LLDB
(lldb) thread step-in
(lldb) step
(lldb) s

Ignore a function when doing a source-level step in

GDB
(gdb) skip abc
LLDB
(lldb) settings set target.process.thread.step-avoid-regexp ^std::|^abc
(lldb) thread step-in -r ^abc
(lldb) sif abc

Source-level single step over

GDB
(gdb) next
(gdb) n
LLDB
(lldb) thread step-over
(lldb) next
(lldb) n

Instruction-level single step in

GDB
(gdb) stepi
(gdb) si
LLDB
(lldb) thread step-inst
(lldb) si

Instruction-level single step over

GDB
(gdb) nexti
(gdb) ni
LLDB
(lldb) thread step-inst-over
(lldb) ni

Step out of the current frame

GDB
(gdb) finish
LLDB
(lldb) thread step-out
(lldb) finish

Return immediately from the current frame

GDB
(gdb) return <RETURN EXPRESSION>
LLDB
(lldb) thread return <RETURN EXPRESSION>

Backtrace and disassemble every time you stop

LLDB
(lldb) target stop-hook add

> bt
> disassemble --pc
> DONE
Stop hook #1 added.

Run until line 12 or current function exits

GDB
(gdb) until 12
LLDB
(lldb) thread until 12

Show the current frame and source line

GDB
(gdb) frame
LLDB
(lldb) frame select
(lldb) f
(lldb) process status

⬡ Breakpoint Commands

Set a breakpoint at all functions named main

GDB
(gdb) break main
LLDB
(lldb) breakpoint set --name main
(lldb) br s -n main
(lldb) b main

Set a breakpoint in test.c at line 12

GDB
(gdb) break test.c:12
LLDB
(lldb) breakpoint set --file test.c --line 12
(lldb) br s -f test.c -l 12
(lldb) b test.c:12

Set a breakpoint at all C++ methods named main

GDB
(gdb) break main
LLDB
(lldb) breakpoint set --method main
(lldb) br s -M main

Set a breakpoint at Objective-C function -[NSString stringWithFormat:]

GDB
(gdb) break -[NSString stringWithFormat:]
LLDB
(lldb) breakpoint set --name "-[NSString stringWithFormat:]"
(lldb) b -[NSString stringWithFormat:]

Set a breakpoint at all Objective-C methods with selector count

GDB
(gdb) break count
LLDB
(lldb) breakpoint set --selector count
(lldb) br s -S count

Set a breakpoint by regex on function name

GDB
(gdb) rbreak regular-expression
LLDB
(lldb) breakpoint set --func-regex regular-expression
(lldb) br s -r regular-expression

Ensure breakpoints work for #include'd .c/.cpp/.m files

GDB
(gdb) b foo.c:12
LLDB
(lldb) settings set target.inline-breakpoint-strategy always
(lldb) br s -f foo.c -l 12

Set a breakpoint by regex on source file contents

GDB
(gdb) shell grep -e -n pattern source-file
(gdb) break source-file:CopyLineNumbers
LLDB
(lldb) breakpoint set --source-pattern regular-expression --file SourceFile
(lldb) br s -p regular-expression -f file

Set a conditional breakpoint

GDB
(gdb) break foo if strcmp(y,"hello") == 0
LLDB
(lldb) breakpoint set --name foo --condition '(int)strcmp(y,"hello") == 0'
(lldb) br s -n foo -c '(int)strcmp(y,"hello") == 0'

List all breakpoints

GDB
(gdb) info break
LLDB
(lldb) breakpoint list
(lldb) br l

Delete / Disable / Enable a breakpoint

GDB
(gdb) delete 1
(gdb) disable 1
(gdb) enable 1
LLDB
(lldb) breakpoint delete 1
(lldb) breakpoint disable 1
(lldb) breakpoint enable 1

◉ Watchpoint Commands

Watch a variable for writes

GDB
(gdb) watch global_var
LLDB
(lldb) watchpoint set variable global_var
(lldb) wa s v global_var

Watch a memory location for writes

GDB
(gdb) watch -location g_char_ptr
LLDB
(lldb) watchpoint set expression -- my_ptr
(lldb) wa s e -- my_ptr

Set a condition on a watchpoint

LLDB
(lldb) watch set var global
(lldb) watchpoint modify -c '(global==5)'
(lldb) c

List all watchpoints

GDB
(gdb) info break
LLDB
(lldb) watchpoint list
(lldb) watch l

Delete a watchpoint

GDB
(gdb) delete 1
LLDB
(lldb) watchpoint delete 1
(lldb) watch del 1

$ Examining Variables

Show arguments and locals for current frame

GDB
(gdb) info args
(gdb) info locals
LLDB
(lldb) frame variable
(lldb) fr v

Show only locals (no args)

GDB
(gdb) info locals
LLDB
(lldb) frame variable --no-args
(lldb) fr v -a

Show contents of local variable bar

GDB
(gdb) p bar
LLDB
(lldb) frame variable bar
(lldb) fr v bar
(lldb) p bar

Show bar formatted as hex

GDB
(gdb) p/x bar
LLDB
(lldb) frame variable --format x bar
(lldb) fr v -f x bar

Show global variable baz

GDB
(gdb) p baz
LLDB
(lldb) target variable baz
(lldb) ta v baz

Show global/static variables in current source file

LLDB
(lldb) target variable
(lldb) ta v

Display argc and argv every time you stop

GDB
(gdb) display argc
(gdb) display argv
LLDB
(lldb) target stop-hook add --one-liner "frame variable argc argv"
(lldb) display argc
(lldb) display argv

Display argc/argv only when stopped in main

LLDB
(lldb) target stop-hook add --name main --one-liner "frame variable argc argv"
(lldb) ta st a -n main -o "fr v argc argv"

Display *this only when stopped in class MyClass

LLDB
(lldb) target stop-hook add --classname MyClass --one-liner "frame variable *this"
(lldb) ta st a -c MyClass -o "fr v *this"

Print array of integers via pointer int *ptr

GDB
(gdb) p *ptr@10
LLDB
(lldb) parray 10 ptr

>_ Evaluating Expressions

Evaluate an expression in the current frame

GDB
(gdb) print (int) printf ("Print nine: %d.", 4 + 5)
(gdb) call (int) printf ("Print nine: %d.", 4 + 5)
LLDB
(lldb) expr (int) printf ("Print nine: %d.", 4 + 5)
(lldb) print (int) printf ("Print nine: %d.", 4 + 5)

Create and assign a convenience variable

GDB
(gdb) set $foo = 5
(gdb) set variable $foo = 5
(gdb) set $foo = (unsigned int) 5
LLDB
(lldb) expr unsigned int $foo = 5

Print ObjC description of an object

GDB
(gdb) po [SomeClass returnAnObject]
LLDB
(lldb) expr -o -- [SomeClass returnAnObject]
(lldb) po [SomeClass returnAnObject]

Print dynamic type of an expression result

GDB
(gdb) set print object 1
(gdb) p someCPPObjectPtrOrReference
LLDB
(lldb) frame variable -d no-run-target someCPPObjectPtrOrReference
(lldb) expr -d no-run-target -- someCPPObjectPtr

Call a function and stop at breakpoints within it

GDB
(gdb) set unwindonsignal 0
(gdb) p function_with_a_breakpoint()
LLDB
(lldb) expr -i 0 -- function_with_a_breakpoint()

Call a crashing function and stop on crash

GDB
(gdb) set unwindonsignal 0
(gdb) p function_which_crashes()
LLDB
(lldb) expr -u 0 -- function_which_crashes()

Show general-purpose registers

GDB
(gdb) info registers
LLDB
(lldb) register read

Write decimal 123 to register rax

GDB
(gdb) p $rax = 123
LLDB
(lldb) register write rax 123

Skip 8 bytes ahead of PC

GDB
(gdb) jump *$pc+8
LLDB
(lldb) register write pc `$pc+8`

Show all register sets

GDB
(gdb) info all-registers
LLDB
(lldb) register read --all
(lldb) re r -a

Show rax, rsp, rbp values

GDB
(gdb) info all-registers rax rsp rbp
LLDB
(lldb) register read rax rsp rbp

Read memory at 0xbffff3c0 — 4 hex uint32 values

GDB
(gdb) x/4xw 0xbffff3c0
LLDB
(lldb) memory read --size 4 --format x --count 4 0xbffff3c0
(lldb) me r -s4 -fx -c4 0xbffff3c0
(lldb) memory read/4xw 0xbffff3c0
(lldb) x/4xw 0xbffff3c0

Read memory at expression argv[0]

GDB
(gdb) x argv[0]
LLDB
(lldb) memory read `argv[0]`
(lldb) memory read --size `sizeof(int)` `argv[0]`

Read 512 bytes of memory and save as text

GDB
(gdb) set logging on
(gdb) set logging file /tmp/mem.txt
(gdb) x/512bx 0xbffff3c0
(gdb) set logging off
LLDB
(lldb) memory read --outfile /tmp/mem.txt --count 512 0xbffff3c0
(lldb) me r -o/tmp/mem.txt -c512 0xbffff3c0

Save binary memory 0x1000–0x2000 to file

GDB
(gdb) dump memory /tmp/mem.bin 0x1000 0x2000
LLDB
(lldb) memory read --outfile /tmp/mem.bin --binary 0x1000 0x2000
(lldb) me r -o /tmp/mem.bin -b 0x1000 0x2000

GDB
(gdb) info proc mappings
LLDB
(lldb) memory region --all
(lldb) me reg --all

GDB

GDB

LLDB
(lldb) disassemble --line

GDB

LLDB

GDB

LLDB
(lldb) image lookup -r -n <FUNC_REGEX>

GDB
(gdb) info line 0x1ec4
LLDB
(lldb) image lookup -v --address 0x1ec4

Look up type Point by name

GDB
(gdb) ptype Point
LLDB
(lldb) image lookup --type Point
(lldb) im loo -t Point

Dump all sections from all images

GDB

LLDB

Dump all symbols

LLDB
(lldb) image dump symtab
